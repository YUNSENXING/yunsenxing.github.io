XRocket - Robot Learning Project

A learning-based control project for vertical rocket landing under stochastic crosswind disturbances, integrating:

  • Imitation Learning (Behavior Cloning & DAgger)
  • Policy Gradient With Bootstrap (PG)
  • Model-Free Reinforcement Learning (PPO)

with a focus on robustness, generalization, and failure-mode analysis across wind strengths and gust frequencies.

KTH DD2410 IROB - Assignment 5 (ROS2 Autonomous Navigation)

ROS2 Foxy project building an autonomous TurtleBot3 navigation pipeline using services and Behavior Trees, with progressive milestones (E / C / A levels).

  • E-Level: Service integration + state machine navigation to goals
  • C-Level: Behavior Tree + sensor integration + dynamic replanning
  • A-Level: Full autonomy with localization (e.g., AMCL), nav stack & exploration

Focus: robust goal handling, recovery behaviors, and resilience to external interruptions (goal changes, deactivation during execution, “kidnapped robot” scenarios).
